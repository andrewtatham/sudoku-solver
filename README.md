# sudoku-solver

A .NET 9 library and console application for parsing and solving Sudoku puzzles using backtracking with constraint propagation (minimum remaining values heuristic).

## Features

- Solves any valid 9×9 Sudoku puzzle
- Supports multiple input formats (see below)
- Fast solver: uses MRV heuristic to minimise backtracking
- Clean library API for embedding in your own projects

## Supported Puzzle Formats

### 1. 81-character string

All 81 cells on one line. Use `1`–`9` for givens and `0` or `.` for empty cells.

```
400050000105076009060400050900000510500090006027000008050004020200360104000010005
```

Using `.` instead of `0` for empty cells is equally valid:

```
4...5....1.5.76..9.6.4...5.9.....51.5...9...6.27.....8.5...4.2.2..36.1.4....1...5
```

### 2. Multi-line grid (9 lines × 9 characters)

```
4...5....
1.5.76..9
.6.4...5.
9.....51.
5...9...6
.27.....8
.5...4.2.
2..36.1.4
....1...5
```

### 3. Grid with separator lines

```
4..|.5.|...
1.5|.76|..9
.6.|4..|.5.
---+---+---
9..|...|51.
5..|.9.|..6
.27|...|..8
---+---+---
.5.|..4|.2.
2..|36.|1.4
...|.1.|..5
```

### 4. SDK format (`.sdk` files)

Lines beginning with `#` are treated as comments and ignored.

```
# Generated by SudokuSolver
# Title: Example Puzzle
4...5....
1.5.76..9
.6.4...5.
9.....51.
5...9...6
.27.....8
.5...4.2.
2..36.1.4
....1...5
```

### 5. Pencilmark / candidate format

Each cell is a space-separated token: a single digit for a given, or multiple digits for candidates (treated as empty), or `.` / `0` for empty.

```
4 . . . 5 . . . .
1 . 5 . 7 6 . . 9
. 6 . 4 . . . 5 .
9 . . . . . 5 1 .
5 . . . 9 . . . 6
. 2 7 . . . . . 8
. 5 . . . 4 . 2 .
2 . . 3 6 . 1 . 4
. . . . 1 . . . 5
```

## Library API

Add a reference to `SudokuSolver.Library` and use the three main classes:

### Parse a puzzle

```csharp
using SudokuSolver.Library;

// From an 81-character string
Grid puzzle = PuzzleParser.Parse("400050000105076009060400050900000510500090006027000008050004020200360104000010005");

// From a multi-line string (any supported format)
Grid puzzle2 = PuzzleParser.Parse(File.ReadAllText("my-puzzle.sdk"));
```

### Solve a puzzle

```csharp
var solver = new Solver();
Grid? solution = solver.Solve(puzzle);

if (solution != null)
{
    Console.WriteLine(solution.ToDisplayString());  // pretty grid
    Console.WriteLine(solution.ToPuzzleString());   // 81-char string
}
else
{
    Console.WriteLine("No solution found.");
}
```

### Work with the Grid directly

```csharp
var grid = new Grid();
grid[0, 0] = 4;   // set row 0, col 0 to 4
int value = grid[0, 0];

bool valid    = grid.IsValid();    // no constraint violations
bool complete = grid.IsComplete;   // all 81 cells filled
bool solved   = grid.IsSolved;     // complete AND valid

int[] row = grid.GetRow(0);        // 9-element array for row 0
int[] col = grid.GetColumn(3);     // 9-element array for column 3
int[] box = grid.GetBox(0);        // 9-element array for top-left 3×3 box

string display = grid.ToDisplayString();  // formatted with separators
string compact = grid.ToPuzzleString();   // 81-char string
string lines   = grid.ToGridString();     // 9 lines × 9 chars
```

### Get candidates for a cell

```csharp
HashSet<int> candidates = Solver.GetCandidates(grid, row: 0, col: 2);
// Returns the set of digits (1-9) that can legally be placed in that cell
```
